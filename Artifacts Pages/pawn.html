Pawn Model

This is a pawn model I made in Blender and rendered using the Eevee render engine. I started this project beacuse, for some time, I wanted to learn how create photorealistic renders, and a pawn model seemed like an easy start to learning.

Even though Cycles could've been used for a better render in terms of this purpose, I chose Eevee for the final render because it was faster and easier to obtain the desired result. Furthermore, creating the pawn helped me better understand how to manipulate shapes using loop cuts and modifiers.
